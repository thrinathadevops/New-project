## 🚀 Quick Start (Literally 5 Minutes)

### 1. Install & Activate
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### 2. Run Dashboard
```powershell
python launcher.py
# Select: 2️⃣ Trading Dashboard
```

### 3. Configure (sidebar)
```
Symbols: RELIANCE,INFY,HDFCBANK
Capital: 20000
Risk %: 2
Confidence: 60%
Refresh: 30s
```

### 4. Execute
- View **Active Trade Opportunities** table
- Click highest confidence trade
- Review entry/stop/target
- Execute in broker platform

**That's it!** ✅

## 📊 What Gets Combined

8 analysis techniques merged into **1 confidence score**:

```
EMA 9/21 (25%) 
+ Price Action (25%)
+ Smart Money (20%)
+ Volume (15%)
+ VWAP (10%)
+ Box Theory (5%)
= UNIFIED CONFIDENCE (0-100%)
```

Example:
- BUY signal where ALL 8 align? → **90-100% confidence** 🎯
- BUY signal where 5 align, 3 conflict? → **60-70% confidence** ⚠️
- Mixed signals? → Filter out (not shown)


```
    ↓

    ↓
Combined Analysis Engine
    ├─ EMA Strategy Analysis
    ├─ Price Action Analysis
    ├─ Smart Money Analysis
    ├─ Box Theory Analysis
    └─ Unified Confidence Calculator
    ↓
Signal Generator (BUY/SELL/HOLD)
    ↓
Trade Plan Builder
    ├─ Entry price
    ├─ Stop loss
    ├─ Target price
    ├─ Position size
    └─ Risk/Reward calculation
    ↓
Display Results + Detai
```
